# Zoom MF Dataflow Animatie Demo

This repository contains a static HTML demo that visualizes the Zoom MF data flow from the input form to live parking-ban and geocoding results.

Main entry point:

- `ZoomMF_Dataflow_Animatie_Demo.html`

## What the demo does

- Reads an address, date, and time from the form.
- Fetches live parking-ban data and geocoding data.
- Shows the filtering steps and the final classification.
- Switches between a technical 2D map and a presentation-oriented 3D map view.
